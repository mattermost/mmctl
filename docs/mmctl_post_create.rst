mmctl post create
-----------------

Create a post

Synopsis
~~~~~~~~


Create a post

::

  mmctl post create [flags]

Examples
~~~~~~~~

::

    post create myteam:mychannel --message "some text for the post"

Options
~~~~~~~

::

  -h, --help              help for create
  -m, --message string    Message for the post
  -r, --reply-to string   Post id to reply to
