mmctl user activate
-------------------

Activate users

Synopsis
~~~~~~~~


Activate users that have been deactivated.

::

  mmctl user activate [emails, usernames, userIds] [flags]

Examples
~~~~~~~~

::

    user activate user@example.com
    user activate username

Options
~~~~~~~

::

  -h, --help   help for activate
