mmctl license remove
--------------------

Remove the current license.

Synopsis
~~~~~~~~


Remove the current license and leave mattermost in Team Edition.

::

  mmctl license remove [flags]

Examples
~~~~~~~~

::

    license remove

Options
~~~~~~~

::

  -h, --help   help for remove
