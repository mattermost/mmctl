mmctl license upload
--------------------

Upload a license.

Synopsis
~~~~~~~~


Upload a license. Replaces current license.

::

  mmctl license upload [license] [flags]

Examples
~~~~~~~~

::

    license upload /path/to/license/mylicensefile.mattermost-license

Options
~~~~~~~

::

  -h, --help   help for upload
