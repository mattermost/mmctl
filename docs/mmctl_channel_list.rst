mmctl channel list
------------------

List all channels on specified teams.

Synopsis
~~~~~~~~


List all channels on specified teams.
Archived channels are appended with ' (archived)'.

::

  mmctl channel list [teams] [flags]

Examples
~~~~~~~~

::

    channel list myteam

Options
~~~~~~~

::

  -h, --help   help for list
