mmctl plugin add
----------------

Add plugins

Synopsis
~~~~~~~~


Add plugins to your Mattermost server.

::

  mmctl plugin add [plugins] [flags]

Examples
~~~~~~~~

::

    plugin add hovercardexample.tar.gz pluginexample.tar.gz

Options
~~~~~~~

::

  -h, --help   help for add

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

::

      --format string   the format of the command output [plain, json] (default "plain")
