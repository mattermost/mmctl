mmctl group team list
---------------------

List team groups

Synopsis
~~~~~~~~


List the groups associated with a team

::

  mmctl group team list [team] [flags]

Examples
~~~~~~~~

::

    group team list myteam

~~~~~~~

::

  -h, --help   help for list
