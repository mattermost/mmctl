mmctl plugin delete
-------------------

Delete plugins

Synopsis
~~~~~~~~


Delete previously uploaded plugins from your Mattermost server.

::

  mmctl plugin delete [plugins] [flags]

Examples
~~~~~~~~

::

    plugin delete hovercardexample pluginexample

~~~~~~~

::

  -h, --help   help for delete
