## mmctl config set

Set config setting

### Synopsis

Sets the value of a config setting by its name in dot notation. Accepts multiple values for array settings

```
mmctl config set [flags]
```

### Examples

```
config set SqlSettings.DriverName mysql
```

### Options

```
  -h, --help   help for set
```
