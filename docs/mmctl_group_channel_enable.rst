mmctl group channel enable
--------------------------

Enables group constrains in the specified channel

Synopsis
~~~~~~~~


Enables group constrains in the specified channel

::

  mmctl group channel enable [team]:[channel] [flags]

Examples
~~~~~~~~

::

    group channel enable myteam:mychannel
