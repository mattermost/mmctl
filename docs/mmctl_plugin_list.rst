mmctl plugin list
-----------------

List plugins

Synopsis
~~~~~~~~


List all enabled and disabled plugins installed on your Mattermost server.

::

  mmctl plugin list [flags]

Examples
~~~~~~~~

::

    plugin list

Options
~~~~~~~

::

  -h, --help   help for list
