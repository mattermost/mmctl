mmctl team restore
------------------

Restore teams

Synopsis
~~~~~~~~


Restores archived teams.

::

  mmctl team restore [teams] [flags]

Examples
~~~~~~~~

::

    team restore myteam

Options
~~~~~~~

::

  -h, --help   help for restore
