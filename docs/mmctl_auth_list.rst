mmctl auth list
---------------

Lists the credentials

Synopsis
~~~~~~~~


Print a list of the registered credentials

::

  mmctl auth list [flags]

Examples
~~~~~~~~

::

    auth list

~~~~~~~

  -h, --help   help for list
