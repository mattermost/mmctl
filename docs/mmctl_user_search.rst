mmctl user search
-----------------

Search for users

Synopsis
~~~~~~~~


Search for users based on username, email, or user ID.

::

  mmctl user search [users] [flags]

Examples
~~~~~~~~

::

    user search user1@mail.com user2@mail.com

Options
~~~~~~~

::

  -h, --help   help for search
