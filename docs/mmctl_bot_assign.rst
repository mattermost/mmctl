mmctl bot assign
----------------

Assign bot

Synopsis
~~~~~~~~


Assign the ownership of a bot to another user

::

  mmctl bot assign [bot-username] [new-owner-username] [flags]

Examples
~~~~~~~~

::

    bot assign testbot user2

Options
~~~~~~~

::

  -h, --help   help for assign
