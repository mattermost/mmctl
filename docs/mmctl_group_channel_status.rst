mmctl group channel status
--------------------------

Show's the group constrain status for the specified channel

Synopsis
~~~~~~~~


Show's the group constrain status for the specified channel

::

  mmctl group channel status [team]:[channel] [flags]

Examples
~~~~~~~~

::

    group channel status myteam:mychannel
