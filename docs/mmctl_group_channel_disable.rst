mmctl group channel disable
---------------------------

Disables group constrains in the specified channel

Synopsis
~~~~~~~~


Disables group constrains in the specified channel

::

  mmctl group channel disable [team]:[channel] [flags]

Examples
~~~~~~~~

::

    group channel disable myteam:mychannel
