mmctl plugin enable
-------------------

Enable plugins

Synopsis
~~~~~~~~


Enable plugins for use on your Mattermost server.

::

  mmctl plugin enable [plugins] [flags]

Examples
~~~~~~~~

::

    plugin enable hovercardexample pluginexample

Options
~~~~~~~

::

  -h, --help   help for enable
