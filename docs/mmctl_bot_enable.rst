mmctl bot enable
----------------

Enable bot

Synopsis
~~~~~~~~


Enable a disabled bot

::

  mmctl bot enable [username] [flags]

Examples
~~~~~~~~

::

    bot enable testbot

Options
~~~~~~~

::

  -h, --help   help for enable
