mmctl config get
----------------

Get config setting

Synopsis
~~~~~~~~


Gets the value of a config setting by its name in dot notation.

::

  mmctl config get [flags]

Examples
~~~~~~~~

::

  config get SqlSettings.DriverName

Options
~~~~~~~

::

  -h, --help   help for get
