mmctl team users remove
-----------------------

Remove users from team

Synopsis
~~~~~~~~


Remove some users from team

::

  mmctl team users remove [team] [users] [flags]

Examples
~~~~~~~~

::

    team remove myteam user@example.com username

~~~~~~~

::

  -h, --help   help for remove
