mmctl bot list
--------------

List bots

Synopsis
~~~~~~~~


List the bots users.

::

  mmctl bot list [flags]

Examples
~~~~~~~~

::

    bot list

Options
~~~~~~~

::

      --all        Optional. Show all bots (including deleleted and orphaned).
  -h, --help       help for list
      --orphaned   Optional. Only show orphaned bots.
