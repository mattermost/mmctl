mmctl bot disable
-----------------

Disable bot

Synopsis
~~~~~~~~


Disable an enabled bot

::

  mmctl bot disable [username] [flags]

Examples
~~~~~~~~

::

    bot disable testbot

Options
~~~~~~~

::

  -h, --help   help for disable
