mmctl group channel list
------------------------

List channel groups

Synopsis
~~~~~~~~


List the groups associated with a channel

::

  mmctl group channel list [team]:[channel] [flags]

Examples
~~~~~~~~

::

    group channel list myteam:mychannel

Options
~~~~~~~

::

  -h, --help   help for list
