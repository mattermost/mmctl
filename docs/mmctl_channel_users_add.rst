mmctl channel users add
-----------------------

Add users to channel

Synopsis
~~~~~~~~


Add some users to channel

::

  mmctl channel users add [channel] [users] [flags]

Examples
~~~~~~~~

::

    channel users add myteam:mychannel user@example.com username
