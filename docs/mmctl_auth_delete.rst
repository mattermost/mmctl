mmctl auth delete
-----------------

Delete an credentials

Synopsis
~~~~~~~~


Delete an credentials by its name

::

  mmctl auth delete [server name] [flags]

Examples
~~~~~~~~

::

    auth delete local-server

~~~~~~~
